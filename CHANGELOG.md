# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-04

### Added
- Mobile-first single-file version for phone, tablet, and foldable usage
- Schedule management grouped by month
- Client search by name, phone, and wedding date
- Regional customs library and custom customs entries
- Execution hub with time flow, staff info, and staff checklist
- Client-linked checklist with in-place add/edit/delete
- Print-style image export for current page, execution hub, and client checklist
- A4-like portrait export layout with long-image support for large content
