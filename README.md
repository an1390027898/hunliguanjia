# 婚礼管家 Mobile

面向手机、平板、翻盖/折叠屏的婚礼执行管理工具。  
这是一个以移动端为核心设计的**单文件版本**，适合婚礼管家、婚礼执行、现场统筹等场景使用。

---

## 项目简介

婚礼管家 Mobile 围绕 **档期、客户、风俗、执行中枢、清单、打印模式导出** 展开，帮助你在手机上快速查看、编辑、跟进和导出婚礼执行内容。

项目特点：

- **移动端优先**：适配手机、平板、翻盖/折叠屏
- **单文件可运行**：核心逻辑集中在 `index.html`
- **本地保存**：浏览器本地存储，刷新后数据仍保留
- **就地编辑**：尽量减少弹窗，适合现场快速修改
- **打印风格导出**：支持清单化、A4纵向风格、高清长图导出

---

## 核心功能

### 1. 首页
- 快速进入主要模块
- 顶部管家名称可直接编辑并自动保存
- 移动端操作更直观

### 2. 档期管理
- 支持新增与编辑档期
- 支持按月份分组查看档期
- 可查看客户、婚期、状态等信息

### 3. 客户管理
- 自动关联档期生成客户信息
- 支持按 **姓名 / 电话 / 婚期** 搜索
- 方便快速定位客户资料

### 4. 风俗管理
- 支持地区风俗库
- 可添加地区
- 可为地区添加风俗内容
- 可根据当前档期自动联动对应地区风俗

### 5. 执行中枢
- 关联当前客户信息
- 包含时间流程清单
- 包含工作人员信息（任务 / 姓名 / 电话）
- 包含工作人员流程清单
- 支持就地添加、编辑、删除

### 6. 清单管理
- 当前客户联动清单
- 按类目分组展示
- 每个类目支持就地新增条目
- 单项支持直接编辑与删除
- 基础模板库支持折叠/收起，减少版面占用

### 7. 导出图片
右上角菜单支持导出：

- 导出当前页图片
- 导出该客户执行中枢图片
- 导出该客户清单图片

导出采用打印模式：

- 更像 **A4 纵向排版**
- **一页一客户**
- 只保留 **标题 + 勾选框 + 备注**
- 内容较多时自动导出为 **高清长图**
- 更适合转发、核对、打印

## 使用方式

### 本地直接打开
1. 下载本项目文件
2. 用浏览器打开 `index.html`
3. 即可开始使用

### GitHub Pages 部署
1. 将本项目上传到 GitHub 仓库
2. 进入仓库 **Settings → Pages**
3. 选择 **GitHub Actions**（推荐）
4. 推送代码后，GitHub 会自动部署

如果你不想用 Actions，也可以直接在 Pages 中选择从主分支部署静态页面。
